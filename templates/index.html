API Usage

You can also use the prediction API directly:

POST /predict
Content-Type: application/json

{
    "features": [5.1, 3.5, 1.4, 0.2]
}

Response:

{
    "predicted_class": 0,
    "predicted_species": "setosa",
    "features_used": {
        "sepal length (cm)": 5.1,
        "sepal width (cm)": 3.5,
        "petal length (cm)": 1.4,
        "petal width (cm)": 0.2
    }
}
